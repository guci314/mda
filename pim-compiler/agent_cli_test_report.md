## 配置要求

各提供商需要设置相应的环境变量：
- OpenAI: `OPENAI_API_KEY`
- DeepSeek: `DEEPSEEK_API_KEY`
- 通义千问: `DASHSCOPE_API_KEY`
- 智谱清言: `ZHIPU_API_KEY`
- 月之暗面: `MOONSHOT_API_KEY`

## 使用建议

1. **国内用户**：推荐使用 DeepSeek、通义千问或智谱清言
2. **国际用户**：推荐使用 OpenAI
3. **成本敏感**：DeepSeek 性价比最高
4. **性能要求**：GPT-4 或 GLM-4 性能最佳
