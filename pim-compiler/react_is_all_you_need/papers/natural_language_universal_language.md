### 3.2 认知科学视角：人类知识的非形式本质

人类知识本质上是：
- **情境依赖的**：同样的概念在不同context下含义不同
- **模糊的**：边界不清晰，这是特性不是缺陷
- **隐喻的**：通过类比和联想构建理解
- **涌现的**：整体大于部分之和

这些特性恰恰是形式化系统无法捕捉的。
